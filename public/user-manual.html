SiliconBari User Manual

Work log, available pages, dynamic sections, backend content areas, and step-by-step management notes for non-technical users.

Purpose

This document works as both a work log and a user manual. It explains what has already been completed, what is available in the website, and how a client or content manager can manage the system from the backend.

Common Tasks

Change Website Logo

  1. Login to /siliconadmin
  2. Open Global Settings
  3. Upload the new logo
  4. Save

Change Favicon

  1. Login to /siliconadmin
  2. Open Global Settings
  3. Upload favicon
  4. Save

Add a New Service

  1. Open Services
  2. Click Add New
  3. Fill the fields
  4. Save

Add a New Industry

  1. Open Industries
  2. Click Add New
  3. Add content and image
  4. Save

Add a New Brand

  1. Open Brands
  2. Click Add New
  3. Upload logo
  4. Save

Add a Testimonial

  1. Open Testimonials
  2. Click Add New
  3. Add client info and comment
  4. Save

Add a Blog or Insight

  1. Open Blogs / Insights
  2. Click Add New
  3. Add thumbnail, title, author, category, short description, and body
  4. Turn on Featured if it should appear on home page
  5. Save

Update SEO for a Page

  1. Open SEO Pages
  2. Find the correct page key
  3. Update meta and OG fields
  4. Save

Future Page Development Notes
